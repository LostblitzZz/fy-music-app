# English 🇺🇸

Hey there! **Fy Music APP** is a premium Discord music bot designed for those who want a smooth, high-quality listening experience without the lag. It's powered by **Discord.js v14**, **@discordjs/voice**, and **yt-dlp** to ensure stability and top-tier audio.

Designed & Developed with ❤️ by **Raitzu**.

## 🌟 Why use Fy Music?

-   **Premium Visuals**: Spotify-inspired embeds with live progress bars and interactive buttons. No more boring text updates!
-   **Everything Supported**: Paste links from **YouTube**, **Spotify**, or **SoundCloud**. It just works.
-   **Autoplay Mode**: Keeps the vibe going by automatically playing related tracks when your queue ends.
-   **Radio Stations**: Built-in 24/7 stations for Lo-Fi, Jazz, K-Pop, and more.
-   **Lyric Finder**: Get the lyrics for your favorite song directly in Discord.
-   **24/7 Mode**: Keep the bot in the voice channel even when it's quiet.
-   **Lossless Feeling**: Optimized audio streaming using `yt-dlp` for the best possible sound quality.

## 🚀 Getting Started

1.  **Clone & Enter**:
    ```bash
    git clone https://github.com/username/fy-music-app.git
    cd fy-music-app
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Setup Environment**:
    Run the setup script to automatically create your `.env` file, then paste your Discord Bot Token inside it.
    ```bash
    npm run setup
    ```
4.  **Let's Play**:
    ```bash
    npm start
    ```

## 📜 Commands

| Command | What it does |
| :--- | :--- |
| `/play` | Plays a song from name or URL |
| `/skip` | Skips to the next track |
| `/stop` | Stops the music and clears everything |
| `/pause` / `/resume` | Toggles playback |
| `/queue` | See what's coming up next |
| `/radio` | Choose a genre-based station |
| `/autoplay` | Toggle automatic recommendations |
| `/247` | Keeps the bot in the channel 24/7 |
